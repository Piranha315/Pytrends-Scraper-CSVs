# Pytrends-Scraper-CSVs
Can be used to scrape Pytrends given an imported CSV and access to a residential proxy network. For those without access to funds for a residential proxy network, many companies offer free trials.


Getting Started: 


pip install pytrends 

pip install requests


Choose a proxy network and enter pertinent info under the 'username', 'password', and 'PROXY_URL' variables in main. Enter the path of your csv.
